Projeto Facebook Sign Up

O que foi desenvolvido

Este foi o primeiro projeto em dupla. Foi proposto pelo time de instrutores que recriássemos a página de cadastro do Facebook de forma mais semelhante possível á original.

Habilidades

    Neste projeto foi verificado a aprendizagem sobre:

  • Formulários em HTML
  • CSS Flexbox para criar layout flexível;
  • Regras CSS específicas para serem aplicadas a dispositivos móveis;
  • Construir páginas que alteram o seu layout de acordo com a orientção da tela,
  • Usar ferramenta de versionamento de código (GIT/GitHub).
